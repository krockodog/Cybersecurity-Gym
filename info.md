# Research Findings: trygit.me Cybersecurity Gymnasium

## Exam Database
- **727 Questions** extracted from official CompTIA PenTest+ sources
  - 460 questions from PT0-002
  - 267 questions from PT0-003 (ExamTopic)
- **5 Domains** with 80/20 analysis
  - Attacks and Exploits (30-33%): 179 questions
  - Planning and Scoping (13-15%): 155 questions
  - Tools and Code Analysis (15-17%): 120 questions
  - Information Gathering and Vulnerability Scanning (21-22%): 100 questions
  - Reporting and Communication (18%): 58 questions
  - General: 115 questions
- **Question Types**: Multiple Choice (711), Multiple Select (8), Hotspot (4), Drag & Drop (4)

## 80/20 Rule per Domain
- **Attacks and Exploits**: SQLi, XSS, Command Injection, Privilege Escalation, Password Attacks, Network Attacks, Wireless, Social Engineering, Metasploit, Lateral Movement
- **Planning and Scoping**: SOW, Rules of Engagement, Compliance (GDPR/HIPAA/PCI), Testing Types (White/Grey/Black Box), Impact Analysis
- **Information Gathering**: Nmap flags, Passive/Active Recon, Vulnerability Scanning, CVSS, Web App Scanning
- **Tools and Code Analysis**: Python/Scapy, PowerShell Empire, BloodHound, Wireshark, Burp Suite, Static/Dynamic Analysis
- **Reporting**: Executive Summary, CVSS in Reports, Remediation, Evidence, Risk Ratings

## PBQ Types (Performance Based Questions)
- Network topology analysis and attack path identification
- Firewall rule analysis and exploitation
- Code review and vulnerability identification
- Log analysis and incident response
- Tool output interpretation (Nmap, Wireshark)
- Scenario-based multi-step exploitation

## Platform Features Required
- AI Teacher personas for each domain
- 80/20 hint system showing key concepts per question
- PBQ visualizations with interactive network diagrams
- TTS audio explanations via ElevenLabs
- Gamification: XP, levels, streaks, badges
- Progress tracking per domain
- Exam simulator with timer
- Dark hacker/cybersecurity theme aesthetic
